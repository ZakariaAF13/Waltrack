## Cara Mengambil Screenshot:

1. Buka aplikasi di browser (http://localhost:5173)
2. Gunakan DevTools untuk responsive mode (mobile view)
3. Ambil screenshot dengan:
   - Windows: Win + Shift + S
   - Mac: Cmd + Shift + 4
   - Browser DevTools: Capture screenshot
4. Simpan dengan nama file sesuai di atas
5. Letakkan di folder `screenshots/`

## Ukuran Rekomendasi:

- Desktop: 1920x1080 atau 1440x900
- Mobile: 375x667 (iPhone SE) atau 390x844 (iPhone 12/13)
- Compress gambar untuk web (max 500KB per file)
